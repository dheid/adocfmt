# Security Policy

## Supported Versions

The following versions of `adocfmt` are currently being supported with security updates:

| Version |     Supported      |
|---------|--------------------|
| 0.1.x   | :white_check_mark: |
| < 0.1.0 | :x:                |

## Reporting a Vulnerability

We take the security of `adocfmt` seriously. If you discover a vulnerability, please do not open a public issue. Instead, please report it privately.

To report a vulnerability, please email **mail@daniel-heid.de**.

Please include:
- A description of the vulnerability.
- Steps to reproduce (a proof-of-concept AsciiDoc file is very helpful).
- The potential impact of the vulnerability.

We will acknowledge receipt of your report within 48 hours and provide a timeline for resolution.
